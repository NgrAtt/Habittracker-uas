
   ```bash

   ```

### Menjalankan Aplikasi
- **Untuk Mobile (Android/iOS)**:
  ```bash
  flutter run
  ```
- **Untuk Web (Chrome)**:
  ```bash
  flutter run -d chrome
  ```
